# Changelog

All notable changes to CyberSage will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-11-01

### Added
- HTTP Request Repeater interface (Burp Suite-like functionality)
- Real-time security vulnerability scanning
- AI-powered vulnerability analysis with Claude AI
- Business impact calculator with ROI analysis
- Automated exploit verification
- Remediation code generation
- WebSocket-based real-time updates
- Modern blue-themed UI with micro-animations
- Comprehensive API documentation
- Docker support for easy deployment
- Session management and request history
- Payload generation tools
- Export functionality (JSON, HTML, PDF)
- Professional documentation and guides

### Changed
- Complete UI/UX overhaul with improved spacing and layout
- Enhanced visual hierarchy for better readability
- Improved error handling and user feedback
- Optimized performance for large datasets
- Updated dependencies to latest versions

### Fixed
- Scrolling issues on various pages
- CORS configuration for cross-origin requests
- WebSocket connection stability
- Memory leaks in long-running sessions
- Database locking issues

### Security
- Implemented rate limiting
- Added input validation and sanitization
- Improved API key management
- Enhanced CORS configuration
- Added security headers

## [1.0.0] - 2024-XX-XX

### Added
- Initial release
- Basic vulnerability scanning
- Simple dashboard interface
- Basic reporting functionality
